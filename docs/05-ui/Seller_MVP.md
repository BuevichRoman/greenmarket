# GreenMarket Seller Cabinet MVP

**Version:** 1.0
**Status:** Normative

## Назначение

Настоящий документ определяет минимальный пользовательский интерфейс продавца GreenMarket Stage 1. Seller Cabinet предназначен исключительно для сопровождения рабочего каталога продавца и публикации его актуальной версии. Продажи, обработка заказов и доставка в первый этап не входят.

## Цель MVP

Seller Cabinet должен обеспечить возможность самостоятельной публикации каталога без участия администратора платформы. После обучения продавец должен самостоятельно: скачать шаблон; заполнить каталог; загрузить новую редакцию; исправить ошибки; повторно опубликовать каталог.

## Общие принципы

Простота; минимальное количество экранов; отсутствие ручного редактирования каталога в браузере; работа исключительно через Publication Service.

## Авторизация

Seller Cabinet доступен только авторизованным продавцам. Каждый продавец имеет доступ исключительно к собственному каталогу.

## Основные функции

Получение шаблона каталога; загрузка Excel; просмотр результата публикации; просмотр истории публикаций; просмотр ошибок последней публикации.

## Экран 1. Главная

Отображает статус продавца, дату последней публикации, текущую версию каталога, количество опубликованных товаров. Если продавец не активирован, отображается соответствующее уведомление.

## Экран 2. Рабочий каталог

Позволяет скачать актуальный шаблон Excel (формат — [Catalog_Template.md](../02-domain/Catalog_Template.md)), ознакомиться с правилами заполнения, перейти к загрузке новой версии каталога. Редактирование каталога в браузере не поддерживается.

## Экран 3. Публикация

Продавец выбирает Excel-файл, запускает публикацию, получает результат обработки. Во время обработки отображается текущий статус публикации (см. состояния в [Publication_FSM.md](../04-services/Publication_FSM.md)).

## Экран 4. Ошибки

При неуспешной публикации отображается список ошибок:

- строка файла;
- колонка;
- код ошибки;
- описание;
- рекомендация по исправлению.

После исправления ошибок продавец выполняет новую публикацию.

## Экран 5. История публикаций

Отображаются версия каталога, дата публикации, результат публикации, количество обработанных товаров. История публикаций доступна только для просмотра.

## Статус продавца

Seller Cabinet отображает текущее состояние продавца: ожидает активации; активен; временно деактивирован. Изменение статуса выполняется только администратором.

## Ограничения

Seller Cabinet не позволяет: изменять Product; изменять ProductGroup; изменять опубликованный каталог вручную; изменять историю публикаций; выполнять модерацию. Все изменения выполняются только посредством новой публикации.

## Источник данных

Seller Cabinet работает исключительно через REST API. Прямой доступ к базе данных запрещён. Все изменения каталога выполняются Publication Service.

## Definition of Done

Seller MVP считается завершённым, если продавец способен самостоятельно: скачать шаблон Excel; подготовить каталог; загрузить новую редакцию; исправить ошибки публикации; успешно опубликовать каталог без участия администратора.
