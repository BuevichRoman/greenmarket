# GreenMarket Glossary

**Version:** 1.0
**Status:** Normative

## Назначение

Настоящий документ определяет основные термины GreenMarket Stage 1. Все остальные документы используют данные определения без повторного описания.

## Unified Catalog

Единый каталог GreenMarket. Содержит дерево товарных групп (ProductGroup) и единый справочник товаров (Product). Не содержит коммерческой информации.

## Working Catalog

Рабочий каталог конкретного продавца. Ведётся продавцом самостоятельно в формате шаблона GreenMarket. Является единственным источником данных для Publication Service.

## ProductGroup

Товарная группа единого каталога. Используется для классификации и навигации. Не содержит коммерческой информации.

## Product

Единая товарная позиция платформы. Используется всеми продавцами. Не содержит цены, остатков, фотографий и других коммерческих характеристик.

## Seller

Зарегистрированный продавец платформы. Продавец публикует рабочий каталог и предоставляет коммерческие предложения покупателям.

## SellerProduct

Коммерческое предложение конкретного продавца. Содержит:

- цену;
- остаток;
- единицу измерения;
- описание;
- фотографии;
- статус публикации.

## CatalogPublication

Запись журнала публикаций рабочего каталога продавца. Создаётся после каждой успешной публикации. После создания не изменяется.

## Publication

Процесс публикации новой редакции рабочего каталога продавца. Включает:

- проверку;
- валидацию;
- сравнение;
- обновление каталога;
- регистрацию публикации.

## Publication Service

Сервис, выполняющий публикацию рабочих каталогов продавцов. Является единственным владельцем данных SellerProduct, SellerProductPhoto и CatalogPublication.

## Catalog Version

Порядковый номер опубликованной версии рабочего каталога продавца. Используется для определения текущей редакции каталога.

## Publication Key

Уникальный идентификатор публикации. Используется для обеспечения идемпотентности Publication Service.

## Active Seller

Продавец, каталог которого доступен покупателям. Активность продавца определяется административным состоянием учётной записи.

## Inactive Seller

Продавец, каталог которого скрыт от покупателей. При этом публикации, товары и история публикаций сохраняются.

## Moderation

Процесс сопоставления SellerProduct с Product либо создания нового Product. Выполняется администратором платформы.

## Buyer Catalog

Каталог, отображаемый покупателям. Формируется автоматически на основании:

- активных продавцов;
- опубликованных SellerProduct;
- единого каталога GreenMarket.

## Stage 1

Первый этап развития GreenMarket. Включает публикацию каталогов и поиск товаров. Не включает оформление заказов, оплату и доставку.

## Stage 2

Следующий этап развития платформы. Включает коммерческие процессы: корзину, заказ, оплату, доставку и дальнейшее развитие платформы.
